# CADFit

This website is inspired by the repositories for the [Nerfies website](https://nerfies.github.io).

**CADFit** is a hybrid optimization-based framework that recovers complex, editable CAD construction sequences from watertight meshes. Rather than predicting programs in a single forward pass, CADFit formulates reconstruction as an explicit IoU-driven optimization over executable CAD programs, supporting extrusions, revolutions, fillets, chamfers, and Boolean union/cut.
